# Projeto Alura

Este repositório reúne exercícios e projetos práticos desenvolvidos durante os estudos de Git/GitHub e lógica de programação com JavaScript na plataforma Alura.

## Objetivo

Organizar e registrar os aprendizados adquiridos em diferentes aulas, mantendo um histórico claro dos projetos e exercícios desenvolvidos.

## Estrutura do repositório

- `git/3386-git-github-projeto_inicial` — projeto inicial utilizando Git e GitHub.
- `Java Script/logica-js-aula_2` — exercícios e atividades de lógica de programação com JavaScript.
- `Java Script/logica-js-aula_3` — continuação dos estudos com novos desafios e implementações.

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript
- Git/GitHub

## Como executar

1. Clone este repositório em sua máquina.
2. Acesse a pasta do projeto desejado.
3. Abra o arquivo `index.html` em um navegador para visualizar a aplicação ou exercício.

## Observações

Este projeto serve como referência para estudos, testes e evolução das habilidades em desenvolvimento web e versionamento de código.
